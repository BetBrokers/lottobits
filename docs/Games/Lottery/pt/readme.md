#Lottobits

Este é o sistema de loterias e jogos de azar que utiliza de forma anônima as compras de bilhetes através do hash de transação na blockchain de criptomoedas, estamos iniciando com loteria e jogo do bicho, mas será possível ter novas implementações de novos jogos automatizados para o sistema com módulos com o decorrer do projeto.

##Como Funciona

O jogador deposita o valor do bilhete no endereço da altcoin do jogo do dia, e cadastra o hash de transação deste depósito no banco de dados junto com as informações do endereço da altcoin para receber o prêmio, caso ganhe o sorteio. 
Com isso é gerado através do hash de transação, uma milhar única do número do bilhete, e assim , o jogo se torna irreversível, tudo feito diretamente com altcoins de forma anônima.
O nosso sorteio acontece todos os dias, e cada sorteio do dia, é estipulado um endereço bitcoin diferente. O ganhador do prêmio receberá diretamente no endereço bitcoin informado no seu perfil.

Apenas transações bitcoin enviadas para o endereço apresentado no jogo que será premiado, portanto o jogador deverá cadastrar o hash tx da transação que ele fez, caso contrário, o sistema não premia.

Rodamos nosso código na deep web, e abrimos para a comunidade auxiliar e poder brincar.

###Regras da loteria
   
Ganha quem acertar:

- Um dos 5 numeros sorteados para os prêmios principais
     
- A milhar, a centena e a dezena de qualquer um dos números sorteados  nos 5 prêmios principais
    
- Bilhetes cujo os números contenham a dezena final idêntica a uma das 3 dezenas anteriores ou das 3 dezenas posteriores à dezena do número sorteado para o prêmio 1, excetuando-se os premiados pela aproximação anterior e posterior.
    
- A unidade do primeiro prêmio.
